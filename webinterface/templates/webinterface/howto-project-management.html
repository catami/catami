Catami HowTo: Manage projects in CATAMI

Important! You'll need to be logged in to perform the following operations

Deleting a Project

Select a project you wish to delete, and click on the red 'Delete' button located on the top left corner of the screen

The following confirmation dialog will appear, click on the confirmation button to proceed with deletion of project

Exporting a project

Select a project you wish to export, and click on the 'Download CSV' button located on the top left corner of the screen

The following confirmation dialog will appear, click on the confirmation button to export the project

Importing a project


On the projects page, click on the 'Import Project' button located on the top left corner of the screen

Click to select csv to import

Select CSV to import

Fill up form with the respective project information and click on "Import" to begin import process.
