Automation, when it works, when it doesn't. How to use it to get work done faster.

Work smarter

The CATAMI annotation tool includes two independent automation tools to help you annotate images more quickly and with less repetition.

1. Image similarity with Dolly

What does it do?

As you annotate images in broad scale annotation the CATAMI annotation tool will find similar images and allow you to copy your annotations to multiple images.

OK, fine. So what is it exactly?

Dolly is an interface between the CATAMI image archive and LIRE (Lucene Image REtrieval) (http://www.semanticmetadata.net/lire/). LIRE includes implementations of common numerical algorithms used widely in this kind of work

How do I use it?

You don't have to do anything, it's already running. The 'Similar Images' tab on the right of the annotation will show you images similar to the one you are looking at in the main view.
