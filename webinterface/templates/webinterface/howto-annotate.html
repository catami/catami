Get started with benthic habitat annotation with CATAMI

About Annotation

CATAMI supports two benthic habitat annotation methods;

  1. Fine scale annotation:

    Assign a number of (x,y) points to an image and individually set each point with the underlying identifying class.

  2. Broad scale annotation:

    Assign classes to an entire image and (optionally) set the percentage coverage of each one.
