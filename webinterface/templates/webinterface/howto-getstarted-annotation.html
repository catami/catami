Catami HowTo: Get started with annotation

How to make a new project and get started with annotation in CATAMI.

Important! You'll need to be logged in to create a project.

1. Create a new project

Head to the projects page and click the '+Create a new project' button, or just hit this button right here:

2. Name your project

Give your project a useful name and (optionally) a description.

3. Select the deployments you with to annotate

Deployments are where CATAMI keeps images organised. You'll need to select one (or more) deployments to have images to annotate.

4. Choose your annotation options

Choose between fine scale annotation (points on images) and broad scale annotation (entire image annotation with percentage cover). Finally, choose how to assign the points (Point sampling) and how to sample the images for annotation (you can skip sampling and choose 'All').
